# Image Browser Based on Gesture Recognition

## How to run

1. Install required modules

   ```shell
   pip install opencv-python
   pip install mediapipe
   pip install pyqt5
   ```

2. Run python script

   ```shell
   python ./picture_viewer.py
   ```
